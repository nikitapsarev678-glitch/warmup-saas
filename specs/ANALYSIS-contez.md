## Страница аккаунтов (/dashboard/accounts)

### Таблица
Колонки: Аккаунт (фото+имя+телефон) | Гео (флаг) | Username | Прокси | Лимиты | Проект | Использ. | Статус | Обновлено | Действия

### Действия на строке
⚙ настройки | 📋 логи | 💬 чаты | 🔥 прогрев | 🗑 удалить

### Хедер
- Кнопка проверки статусов всех аккаунтов (подключается к Telegram и проверяет каждый)

- Импорт аккаунтов | + Подключить аккаунт

## Детальный дравер аккаунта (Sheet с 5 табами)

### Таб: Информация
- Статус аккаунта (зелёный блок: Активен + Авторизован)
- Grid 2x2: Username | Телефон | Подключён (дата) | Активность (дата)
- Тип подключения (Телефон / Session)
- ID аккаунта (UUID)
- Секция "Чат-боты"

### Таб: Рассылки (лимиты per-account)
Stats блок (4 карточки с цветами):
- 🎯 Рассылки: 0 / всего участий
- ✈ Сообщений: 0 / всего отправлено
- ↗ Успешность: 0% / успешных/ошибок
- ⏰ Сегодня: 0 / без лимита

Настройки лимитов:
- **Дневной лимит рассылки** (0 = без лимита): пресеты [Без лимита, 50, 100, 200, 500, 1000]
- **Сброс лимитов каждые**: [6ч, 12ч, 24ч, 48ч]
- **Лимит рассылки в час**: [10, 20, 30, 50, 100] — защита от спам-блока
- **Лимит на одну группу/канал**: [1, 3, 5, 10, 20]
- **Лимит рассылки в ЛС** (личные сообщения): [Без лимита, 10, 20, 50, 100, 200]
- **При достижении лимита** (radio):
  - 🔴 Поставить на паузу (до сброса лимита)
  - ⬜ Заменить на другой аккаунт
  - ⬜ Увеличить задержку (x2.0)
- **Дневной лимит инвайтинга**: [Без лимита, 5, 10, 20, 50, 100]
- **Дневной лимит отметок в сторис**: [Без лимита, 5, 10, 20, 50, 100]
- **Дневной лимит парсинга**: [Без лимита, 3, 5, 10, 20, 50] групп/день
- Кнопка **Сохранить лимиты**

### Таб: Профиль
- Поля: Имя, Фамилия, Username (валидация: латиница, цифры, подчёркивания, мин 5 символов)
- О себе Bio (textarea, счётчик 40/70)
- Дата рождения (День/Месяц/Год dropdowns)
- Создать пароль 2FA (зелёный блок, рекомендуемый):
  - Input пароля (+ глаз)
  - Input подсказки (опционально)
  - Кнопка "+ Создать 2FA пароль"
- Аватар профиля (текущий + кнопка Загрузить аватарку, до 5МБ)
- Кнопка **Сохранить изменения**

### Таб: Прокси
- Info блок (фиолетовый): объяснение прокси
- Toggle "Использовать прокси" (текущее состояние: Прямое соединение)
- Кнопка **Сохранить настройки прокси**

### Таб: Пауза
- Статус блок (зелёный если активен): "Аккаунт активен / Пауза не установлена"
- Проверка @SpamBot (кнопка Проверить, статус: Статус неизвестен)
- Объяснение паузы (авто при PEER_FLOOD каждые 20 мин, лимит, спамблок)
- Установить паузу вручную: Input + пресеты [1ч, 6ч, 12ч, 24ч, 48ч, 72ч]
- Кнопка **Поставить на отлёжку** (оранжевая)
- Блок "Автоматическая отлёжка": при лимите, спамблоке, PEER_FLOOD/FLOOD_WAIT

### Таб: Автопрогрев (per-account фоновые действия)
ОСНОВНЫЕ ДЕЙСТВИЯ:
- **Просмотр диалогов** (toggle синий): Диалогов за сессию: [input]
- **Реакции в группах** (toggle розовый): Реакций в день: [input]
- **Просмотр сторис** (toggle синий): Сторис за сессию: [input], кнопка "Просмотреть сторис сейчас"

РАСШИРЕННЫЕ ДЕЙСТВИЯ:
- **Скролл каналов** (toggle синий): Каналов за сессию, Постов от, Постов до
- **Реакции на посты** (toggle розовый): Реакций в день, Выбор эмодзи (клик по иконкам: 👍❤️🔥👏😊🎉😍🤔💯👀🙏💪)
- **Чтение комментариев** (toggle зелёный): Комментов за сессию
- **Рандомизация времени**: ± N минут
- **Лог действий** (пустой если не было) + кнопка Обновить
- **Автовступление в каналы** (toggle синий): Каналов для вступления, кнопка "+ Вступить в подборку каналов"
- Info блок "Как работает автопрогрев": случайное время, рандомные интервалы, нагрузка 10000+ аккаунтов
- Кнопка **Сохранить настройки** (оранжевая)

## Страница прогрева (/dashboard/warming)

### Список задач
- Заголовок: "Прогрев / AI-общение для имитации активности"
- Кнопки: ↻ Обновить | + Добавить
- Табы: Telegram | WhatsApp
- Stats: N Задач | N Актив. | N Готово | N Ошибок

### Создание задачи прогрева (модалка с 3 табами)

**Таб: Общие**

1. Название задания (авто: "Прогрев ДД.ММ ЧЧ:ММ")
2. Проект (dropdown)
3. Выберите аккаунты (table: Аккаунт | иконки статусов | дней | настройки)
4. **ИИ Прогрев** блок: [Уникальные] [Контекст] — тип генерации диалогов
5. **Тема общения** (card-select, несколько):
   - 🧑 Повседневная жизнь (планы, покупки, еда, погода)
   - 💼 Работа и дела (задачи, проекты, встречи)
   - 🎮 Хобби и увлечения (спорт, кино, музыка, путешествия)
   - 💬 Свободное общение (шутки, мемы, мнения)
6. **Режим взаимодействия**:
   - 👥 1к1 (Аккаунты в парах)
   - ⚡ 1ко N (Все рандомно)
7. **Типы контента** (checkboxes): ✓Текст, ✓Эмодзи, Картинки, Видео, Голосовые, Аудио, Файлы
8. **Пауза между сообщениями** (radio):
   - Моментально (2-5 сек)
   - Быстро (5-15 сек)
   - ✓ Среднее (15-45 сек)
   - Медленно (45-90 сек)
   - Очень медленно (2-5 мин)
   - Максимально (5-10 мин)
   - Своё время (от/до сек custom inputs)
9. **Лимит на аккаунт**: [input] сообщений на аккаунт
   Пресеты: [10→3дня, 20→7дней, 35→14дней, 50→30дней]
10. **Диалогов в день**: [input] (пресеты: 5мало, 10норма, 20много, 30макс)
    Подсказка: "Меньше диалогов = более естественное поведение"
11. **Серия сообщений**: Минимум [1] — Максимум [3] сообщений подряд
12. **Пауза между парами**: Мин [30] — Макс [120] сек
    Рекомендация: 30-120 сек для естественного поведения
13. **Ответы на сообщения** (toggle фиолетовый): % ответов slider (1-100%, по умолчанию 25%)
14. **Удаление сообщений** (toggle): случайно удалять некоторые
15. **Когда запустить**: [⚡ Сейчас] [📅 По расписанию]

**Таб: Расписание**
- Планировщик toggle (Выкл по умолчанию)

**Таб: Доп. функции**
- Реакции в диалогах (toggle)
- Комментинг (toggle) — комментарии в каналах
- Внешние реакции (toggle) — реакции в других чатах
- Свой канал (toggle) — ИИ публикует посты
- Публикация сторис (toggle) — требует Telegram Premium

**Нижняя панель (sticky)**:
- 📅 Дней прогрева: [3] [7] [14] [30]
- Стоимость: 200 🪙 / Баланс: 400 🪙
- Кнопка [Купить токены] (оранжевая)
- [Отмена] [✓ Сохранить]

## Монетизация (ключевое открытие!)

**Contez использует ГИБРИДНУЮ модель:**
1. **Подписка** — даёт доступ к функциям и лимит аккаунтов
2. **Токены** — расходуются на конкретные операции (ИИ прогрев = 200 токенов)

Токены покупаются отдельно (кнопка "Купить токены").
Баланс токенов показывается в нижней панели при создании задачи.

## Что НЕ было в нашем плане и нужно добавить

### Критично (без этого продукт неполный):
1. **Токен система** — отдельная таблица `tokens`, покупка пакетов, списание при создании задач
2. **Автопрогрев per-account** — настройки в детале аккаунта, фоновый воркер читает их
3. **ИИ Прогрев task** — AI-генерация диалогов через OpenRouter, темы, режимы, типы контента
4. **Прокси менеджер** — отдельная сущность, привязывается к аккаунтам
5. **Аккаунт лимиты** — per-account лимиты (рассылки, инвайтинг, парсинг, сторис)
6. **SpamBot проверка** — запрос через Telethon к @SpamBot
7. **Пауза/отлёжка** — ручная и авто, per-account
8. **Импорт via TData/Session+JSON ZIP** — bulk import

### Желательно (для конкурентности):
9. **Тема/контекст диалогов** — выбор темы для ИИ
10. **Лог действий** per-account (модалка с табами)
11. **Проверка всех аккаунтов** (batch status check)
12. **2FA управление** из интерфейса
